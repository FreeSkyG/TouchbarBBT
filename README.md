# TouchbarBBT

> 主意在美化 Macbook pro 上的 Touchbar，其变的更好用，更好看



## 所需要的工具和材料

- [BetterTouchTool](https://folivora.ai/) （系统增强工具）
- Sketch （设计软件）

获取方式我就不多言了，有PJ版的



## 实现方式
`BetterTouchTool` 里面有个针对 Touch bar 自定义的模块，里面可以完全自定义其功能和图标。


## 本资源库包含的内容

 - 一些自定义的图标
 - 自己在用的一个方案
